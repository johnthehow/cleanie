# Cleannie 小干净
## Designing Principle 设计理念
* 白名单式筛选合法token, 合法保留, 非法剔除

## Features 功能介绍
* 适用于一行一句型平白文本
* 替换自定义符号
* 去除行号
* 转换HTML字符实体为正常文字
* 替换非ASCII标点符号
* 删除多余空格
* 删除空行
* 空格分词
* 有注音符号的字母转无注音符号
* 统计不合法字符
* 统计字母-数字-符号组合占比
* 拆分标点符号连接的单词
* 清除单词两边的非发声符号
* 替换URL为[URL]
* 小写化
